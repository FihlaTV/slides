# *Let's Get Started*
### How to get tQuery * download .zip on github [here](https://github.com/jeromeetienne/tquery/downloads) * unzip it * point a browser to it
### Which Browser to use * a browser with WebGL! * preferably [chrome](http://www.google.com/chrome) * Thus we reduce version issues * possible alternatives: firefox, opera
### Point Your browser to tQuery #### Using file access ``` $ open -a "/Applications/Google Chrome.app" --args --disable-web-security --allow-file-access-from-files ``` #### Using http access ``` $ cd tQuery $ python -m SimpleHTTPServer ``` then goto [http://127.0.0.1:8000](http://127.0.0.1:8000) with your browser

# *Little Bit of Background*
### What is WebGL * Browser stuff * HTML5 Standard * Copied from OpenGL 2.0 ES * Talk directly to graphics cards * Very efficient - way above normal web stuff * Quite verbose tho
### WebGL Support * Rather new thing * Modern Browsers got support - IE9 hasn't * Desktop support: *Ok* * Mobile support: *Not Yet* * Very marginal * IOS can do it but [not by default](http://learningthreejs.com/blog/2012/05/29/three-js-sportcar-on-ios/) * firefox/opera mobile
### Libraries Needed * Make it easier/shorter to code * [Three.js](https://github.com/mrdoob/three.js/) is very popular * We gonna use it thru [tQuery](http://jeromeetienne.github.com/tquery/) * a jQuery-like API for THREE.js
### What is tQuery * thin library on top of three.js * three.js 100k - tQuery 8k * Make it easier/shorter to code * all opensource - under [mit license](https://github.com/jeromeetienne/tquery/blob/master/MIT-LICENSE.txt) * a jQuery-like API for three.js * Even Easier to learn if you already know jQuery * *85% of webdevs* already know jQuery
### and Javascript ? * Programming language on the Web * May be hard to get - [wtfjs](http://wtfjs.com/) * Dont worry we will do that together :)
