## *WebGL is Low Level*
### WebGL is Low Level #### Origin * designed for hardware accelerated graphic * Talk to the hardware directly #### Fitness ? * Uses a logic foreign to common WebDevs * Usual Web API are far more abstract * cf. canvas2D
### WegGL API Status * WebGL is Old * WebGL is Low Level * WebGL is Verbose => *Not Suitable for Webdevs Culture*
